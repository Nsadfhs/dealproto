Let's talk DPT

Dealprotocol은 Private API(Application Programming Interface)를 통하여 게임 개발사가 Smart Contract 를 통하여 Blockchain 상에 쉽게 게임을 런칭할수 있도록 Dealprotocol를 제공 합니다.

DealChain - API

DealChain은 JSON-RPC 또는 SHA-3 암호화를 이용한 Private API를 이용하며, DApp이나 SDK의 통신은 Public API로 이루어집니다.

verified_user

DealChain - SDK

Dealprotocol 를 이용하여 게임개발사들은 Android, iOS, Unity, Cocos 등 플랫폼에 구애 받지 않고 블록 체인에 대한 노하우가 없이 SDK를 이용하여 개발을 쉽게 할 수 있습니다.

DPT Wallet

Dealprotocol에서는 통합 지갑인 ‘DPT Wallet’를 제공하며, 모든 게임에 사용되는 통합계정과 DPT Wallet를 제공받을 수 있습니다.

DPT Wallet

지갑을 통해 게임 아이템의 구매, 토큰의 거래 및 스왑과 같은 기능을 사용할 수 있습니다.

‘DPT Wallet은 게임플레이어들의 모든 토큰을 조회 및 자동 스왑기능을 제공하며,
지갑내 개인의 자산들은 거래소나 마켓을 통하여 거래될 수있습니다.
DPT Platform 생태계에서 게임 개발사들은 API, SDK를 통하여 손쉽게 지갑과 게임의 연동이 가능하며 개발회사의 자체 지갑를 보유할 수 있습니다.
